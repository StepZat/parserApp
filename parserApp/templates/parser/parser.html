HH.ru Parser

Добро пожаловать в программу парсинга сервиса Headhunter!

Для использования парсера, пожалуйста, авторизируйтесь по кнопке Login, расположенной слева.

Если же вы еще не зарегистрированы, сделайте это прямо сейчас :)

  • Парсер работает только с вакансиями, выложенными на территории Российской Федерации;
  • Для того, чтобы обновить список регионов, направлений деятельности и других словарей, воспользуйтесь кнопкой Обновить список регионов;
  • Для того, чтобы выполнить поиск по вакансиям, воспользуйтесь кнопокой Парсер. Имейте ввиду, что обязательным фильтром является регион;
  • Для того, чтобы вернуться к последнему запросу, воспользуйтесь кнопкой Просмотреть последний запрос;
  • Для того, чтобы выполнить выгрузку сохраненного поиска, воспользуйтесь кнопкой Сохранить как CSV.
